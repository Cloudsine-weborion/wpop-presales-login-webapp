Web shells

Overview

Web shells provision a backdoor and establish persistent access to a system. It is a script that is injected within an accessible server. The shell allows a set of functions to be executed to leak or discover senstive information of the system. These attacks are also commonly known as Remote Code Execution

RCE is a security vulnerability that allows an attacker to execute arbitrary code on a remote system. Attackers exploit vulnerabilities in software or services, such as web applications, APIs, or network services.
